# AgriFlux - AI-Driven Sustainable Agriculture Intelligence

AgriFlux is a production-grade intelligent platform designed for sustainable agriculture. It leverages AI, machine learning, and climate intelligence to help farmers increase crop yield while optimizing resources.

## 🚀 Features
- **AI Yield Prediction**: Predict crop yield based on soil NPK, weather, and historical data.
- **Smart Irrigation**: Precision scheduling based on crop needs and rainfall forecasts.
- **Soil Health Advisor**: AI-powered analysis with fertilizer recommendations.
- **Climate Risk Forecasting**: Real-time weather intelligence and seasonal risk scoring.
- **Satellite Monitoring**: NDVI/EVI vegetation index monitoring via interactive maps.
- **Climate Simulator (🏆 Hackathon)**: Simulate future climate scenarios and their impact.
- **Crop Switching Advisor (🏆 Hackathon)**: Get climate-resilient alternative crop suggestions.
- **Sustainability Score (🏆 Hackathon)**: Track and improve your farm's environmental impact.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, TypeScript, TailwindCSS, Recharts, Lucide React, i18next.
- **Backend**: Node.js, Express, TypeScript, JWT, Axios, Dotenv.
- **AI/ML**: Python (Scikit-learn, XGBoost integration), Rule-based fallback engine.

## 📦 Getting Started

### 1. Frontend
```bash
cd Frontend
npm install
npm run dev
```

### 2. Backend
```bash
cd Backend
npm install
npm run dev
```
